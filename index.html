Concept of LDU Factorization

LDU Factorization is a decomposition of a square matrix A into three simpler matrices.

  • L = Lower triangular matrix
  • D = Diagonal matrix
  • U = Upper triangular matrix
  • A=L⋅D⋅U
  • Used to solve linear equations
  • Simpler way to find inverse of matrix.

    • We have created it only for matrix of order 2 and order 3.
